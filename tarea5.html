Hosting y Dominios

Que es un hosting

El Hosting o alojamiento web es el servicio que guarda los archivos de una pagina web en un servidor para que pueda verse en internet las 24 horas.Es como "La casa"donde vive tu sitio Web.

Que es un Dominio

El dominio es el nombre o direccion de una pagina web,por ejemplo:mi pagina web.com Es lo que las personas escriben en el navegador para entrar a un sitio web
